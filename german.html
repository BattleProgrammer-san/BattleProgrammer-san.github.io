Kurzerklärung

Dies ist ein simpler Währungsrechner.

Auf dieser Seite kann man sehen, wie einfach es ist Sprache und Datenset anzupassen. Die Initialisierung des W#hrungsrechners kann um optionen ergänzt werden. Auf dieser Seite wurden die folgenden Optionen verwendet: 


    options = {
        "localePath" : "/overrideExamples/de-DE.json",
        "overrideCurrencies": "/overrideExamples/exchangeRates.json",
        "showCurrencySymbol": false
    }
    cConverter.init("injectHere", options);

Alle Optionen sind optional. Die Übersetzungen und neuen Währungen können überschrieben werden. Der boolsche Parameter "showCurrencySymbol" kann wie hier deaktiviert werden, um das Währungssymbol im Ergebnis auszublenden.

Außerdem wurde auf dieser Seite das Styling per inline Style überschrieben. Dazu sollte die Klasse 


    .cConverterCustomStyle.cConverter__container

verwendet werden.

Das Grundbeispiel kann hier gefunden werden.
