Currency Converter

Quick Explanation

This is a simple currency converter. It is made for FactSet.

The converter itself is modular and can be injected on any website with just a few lines of code. To do that, you need to add the folder "cConverter" to the root of your website. Afterwards, add the following code: 


    <script src="cConverter/cConverter.js"></script>
    <script>cConverter.init("injectHere");</script>

The code above will inject the converter into the element with the id "injectHere". Additionally, a few options regarding locale and currencies can be overwritten. An example for that can be found here.
